## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohdanas86/intern
   ```

2. **Navigate to the project directory:**
   ```bash
   cd intern
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Start the project:**
   ```bash
   npm start
   # or
   yarn start
